Para habilitar la tabla precios_servicios en Supabase, ejecutá el SQL que está más abajo en el tutorial.

SQL para tabla de precios — ejecutar en Supabase

CREATE TABLE IF NOT EXISTS precios_servicios (
  id        SERIAL PRIMARY KEY,
  servicio  TEXT NOT NULL,
  slug      TEXT UNIQUE,
  venta     NUMERIC(10,2) DEFAULT 0,
  costo     NUMERIC(10,2) DEFAULT 0,
  moneda    TEXT DEFAULT 'ARS',
  es_combo  BOOLEAN DEFAULT false,
  descripcion TEXT,
  creado_en TIMESTAMPTZ DEFAULT NOW()
);
ALTER TABLE precios_servicios
  DISABLE ROW LEVEL SECURITY;

  • Listame los clientes vencidos
  • ¿Cuántos clientes tengo por servicio?
  • ¿Cuándo tengo que renovar con mis proveedores?
  • ¿Qué correos están asignados a más de un cliente?

Cómo usar los precios de servicios

  • Andá a "Precios de servicios" en el menú
  • Creá un precio por cada servicio que vendés
  • Para combos (ej: Netflix + Disney) marcalos como combo
  • Al crear un cliente nuevo podés elegir el precio del listado
  • El estimativo financiero usa estos precios automáticamente

Filtros de clientes

  • Buscar por nombre, teléfono o correo
  • Filtrar por país (Argentina / Uruguay)
  • Filtrar por estado (Activo / Inactivo)
  • Filtrar por servicio contratado
  • Filtrar por vencimiento (vencidos, hoy, 7 días, etc.)
  • Ordenar por nombre, vencimiento, días restantes, precio
